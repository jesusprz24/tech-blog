# Description
Developers will be allowed to make, view and edit posts, Leave comments, log in and register. 
This Tech Blog uses MVC, SQL, Handlebars, Node.js and Express.js.

# Installation
1. Clone the repo into the terminal
2. This application requires the use of Node.js, Express, Handlebars, and MySQL. Please make sure those are installed.
3. Next install the npm packages
- npm p init
- npm i bcrypt
- npm i connect-session-sequelize
- npm i dotenv
- npm i express-handlebars
- npm i inquirer
- npm i sequelize
